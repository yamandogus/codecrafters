# CodeCrafters Web

Bu proje, CodeCrafters platformunun frontend (ön yüz) uygulamasıdır. [Next.js](https://nextjs.org) ile geliştirilmiştir.

## Başlarken

Geliştirme sunucusunu çalıştırmak için aşağıdaki komutlardan birini kullanabilirsiniz:

```bash
npm run dev
# veya
yarn dev
# veya
pnpm dev
# veya
bun dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açarak uygulamayı görebilirsiniz.

Sayfayı düzenlemeye `app/page.tsx` dosyasından başlayabilirsiniz. Dosyayı düzenlediğinizde sayfa otomatik olarak güncellenecektir.

## Teknoloji Yığını

- **Framework:** Next.js
- **UI Kütüphanesi:** React
- **Stil:** Tailwind CSS (Varsayılan Next.js yapılandırması)
- **Font:** [Geist](https://vercel.com/font) (Next.js font optimizasyonu ile)
